sd2.sdlttf - SDL2_ttf library wrapperÂ¶

The sdl2.sdlttf module is a ctypes-based wrapper around the SDL2_ttf library. It wraps nearly all publicly accessible structures and functions of the SDL2_ttf library to be accessible from Python code.

A detailed documentation about the behaviour of the different functions can found within the SDL2_ttf documentation.
