sdl2.util.get_cfg(section, option, convert=False, path=None)¶

Interface for ConfigParser.

Parameters:
  • section (str) – section in the cfg file (e.g. “MANAGER” == [MANAGER])
  • option (str) – key inside the group (e.g. “SCREEN_WIDTH”)
  • convert (bool) – if True the value obtained will be converted using :py:meth: ast.literal_eval before its returned.

Example

>>> get_cfg(section='MANAGER', option='SCREEN_WIDTH', convert=True)
1024
Returns:str (value of the cfg entry, if a match is found) None (if the option/key is not found)
sdl2.util.iter_nested(iterables)¶

Iterate over a iterable of depth one or two.

In both cases yield one item at a time.

Parameters:iterables (iterable) – a iterable or a iterable of iterables.

Examples

>>> list(nested_or_not(((0, 0, 10, 10), (20, 20, 10, 10))))
[(0, 0, 10, 10), (20, 20, 10, 10)]

>>> list(nested_or_not((30, 30, 10, 10)))
[(30, 30, 10, 10)]

# in both cases this would work: # for tup in nested_or_not(): # do something
